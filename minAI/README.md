# Nano: the Gathering

This is a vibe-coded webpage of the imaginary _Nano: the Gathering_ card game
inspired by _Magic: the Gathering_.

## AI Prompt Log

See [PROMPTS.md](PROMPTS.md) for a list of prompts.

### Prompts that worked well

- the prompts that included important context such as sample data or the file structure of the project
- the prompts that listed multiple features worked surprisingly well

### Prompts that failed horribly

No prompt was an utter failure. However, sometimes it would miss something when we listed multiple features in the same prompt (e.g., illegible text in light mode). Other times it would backtrack, for instance, with an unclosable modal that it had removed earlier but came back again.

### What we gained by using AI for generation

_Rapid_ development of something that looked fairly decent, all things considered.

### What we lost by using AI for generation

We have no comprehension of the code. We may be able to dissect and read it, but as such, we have no incentive to understand any of it.
